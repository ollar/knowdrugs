# KnowDrugs #

This is a https://knowdrugs.info React Native app.

### How do I get set up? ###

* git clone
* `npm i` / `yarn`
* react-native run-android / run-ios

### Contribution guidelines ###

* Code review
* React proptypes
* pure components when available
* eslint airbnb
* immutable stores
